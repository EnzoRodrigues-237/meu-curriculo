# Currículo Online

Este repositório contém meu currículo online, desenvolvido com HTML e CSS e publicado via GitHub Pages.

O objetivo do projeto é apresentar meu perfil profissional de forma clara, organizada e acessível, reunindo informações sobre formação, habilidades técnicas, projetos e contatos.

## 🚀 Acesso

Você pode visualizar o currículo diretamente pelo GitHub Pages:
[Currículo GitHub - Enzo Rodrigues de Oliveira Silva](https://enzorodrigues-237.github.io/meu-curriculo/)

## 📁 Estrutura do projeto

- `index.html` — estrutura da página
- `style.css` — estilização e responsividade
- `README.md` — documentação

## 🧠 Tecnologias utilizadas

- HTML5
- CSS3
- Google Fonts
- Font Awesome

## 🎯 Objetivo

Criar uma página de currículo com identidade visual moderna, navegação simples e apresentação profissional, servindo como apoio em processos seletivos e portfólio pessoal.

## 💼 Conteúdo apresentado

A página inclui:

- Objetivo profissional
- Competências técnicas
- Formação acadêmica
- Projetos relevantes
- Informações de contato

## 🧩 Projeto em destaque

### Arena E-Sports — Plataforma de Gestão de Campeonatos

Projeto em desenvolvimento com arquitetura separada entre front-end e back-end, voltado ao gerenciamento de campeonatos e eventos de e-sports.

**Principais funcionalidades:**
- Sistema de autenticação de usuários
- Gerenciamento de campeonatos
- Inscrição e controle de participantes
- Estrutura inicial para chaveamento de torneios

**Tecnologias:**
- React
- FastAPI
- PostgreSQL
